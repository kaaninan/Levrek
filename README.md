# Levrek

## Levrek, Simple GitHub-CLI
[![npm version](https://badge.fury.io/js/levrek.svg)](https://badge.fury.io/js/levrek)

Minimalistic GitHub Command Line Interface


### Installation


```sh
$ npm install -g levrek
```

### Usage

Easy-to use

```sh
$ levrek -u <username>
```

Short by last updated time
```
$ levrek -u <username> -s
```

Like:
```sh
$ levrek -u omerraker
```
